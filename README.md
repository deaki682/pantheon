# Pantheon

Autonomous AI trading system — three agents (Oracle, Achilles, Delphi)
sharing one Robinhood account, each running a distinct strategy across
different time horizons.

The full implementation lives on `claude/ai-trading-gods-cbw1uh`.
Operational state lives on `claude/live` (separate branch managed by
`pantheon/persist.py`).

See the open PR for code review.
