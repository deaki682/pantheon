# MetaStrip — Privacy

**MetaStrip collects nothing, stores nothing, and sends nothing.**

- Your photos are read into memory only to process them, and are never uploaded,
  saved, or transmitted anywhere. There is no server to receive them.
- MetaStrip requests **no permissions** and **no access to any website**. It can
  only ever see the files you explicitly drag into its page.
- No accounts, no sign-in, no cookies, no local storage of your data, no
  analytics, no telemetry, no error reporting, no third-party code.
- It works with your network connection turned off. That's the proof: with no
  network, the files cannot go anywhere.

Because MetaStrip does not collect, use, store, or transmit any user data, no
personal information is ever at risk through this tool.
